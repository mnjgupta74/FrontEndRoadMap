DOM मैनिपुलेशन और एवेंट्स

1. DOM क्या है?

DOM (Document Object Model) HTML दस्तावेज़ का एक प्रोग्रामिंग इंटरफेस है। यह वेब पेज को एक ट्री स्ट्रक्चर के रूप में रिप्रेजेंट करता है जिसमें:

  • एलिमेंट्स नोड्स बनते हैं
  • टेक्स्ट टेक्स्ट नोड्स बनते हैं
  • एट्रिब्यूट्स एट्रिब्यूट नोड्स बनते हैं

2. DOM एलिमेंट्स को सेलेक्ट करना

मुख्य सेलेक्टर मेथड्स:

// ID से सेलेक्ट करना
const element = document.getElementById('myId');

// क्लास से सेलेक्ट करना
const elements = document.getElementsByClassName('myClass');

// टैग नेम से सेलेक्ट करना
const elements = document.getElementsByTagName('div');

// CSS सेलेक्टर से सेलेक्ट करना
const element = document.querySelector('.myClass');
const elements = document.querySelectorAll('.myClass');

उदाहरण:

// HTML
<div id="demo" class="box">Hello World</div>

// JavaScript
const demoElement = document.getElementById('demo');
const boxElements = document.getElementsByClassName('box');
const allDivs = document.getElementsByTagName('div');
const firstBox = document.querySelector('.box');

3. DOM मैनिपुलेशन

एलिमेंट्स को मॉडिफाई करना:

// कंटेंट बदलना
element.textContent = 'नया टेक्स्ट';
element.innerHTML = '<strong>बोल्ड टेक्स्ट</strong>';

// एट्रिब्यूट्स बदलना
element.setAttribute('class', 'newClass');
element.classList.add('newClass');
element.classList.remove('oldClass');

// स्टाइल बदलना
element.style.color = 'red';
element.style.backgroundColor = 'blue';

उदाहरण:

यह एक डेमो बॉक्स है

const box = document.getElementById('demoBox');
box.style.backgroundColor = '#f0f0f0';
box.querySelector('p').textContent = 'टेक्स्ट बदल गया!';

4. एवेंट हैंडलिंग

एवेंट्स को लिसन करना:

// बेसिक एवेंट लिसनर
element.addEventListener('click', function(event) {
    console.log('क्लिक हुआ!');
});

// एरो फंक्शन के साथ
element.addEventListener('mouseover', (event) => {
    console.log('माउस ऊपर आया!');
});

// एवेंट को रिमूव करना
const handler = function() {
    console.log('एक बार ही चलेगा!');
};
element.addEventListener('click', handler);
element.removeEventListener('click', handler);

उदाहरण:

क्लिक काउंट: 0

const button = document.querySelector('#eventDemo button');
const countSpan = document.getElementById('clickCount');
let count = 0;

button.addEventListener('click', () => {
    count++;
    countSpan.textContent = count;
});

5. कॉमन एवेंट्स

  • माउस एवेंट्स:
    • click
    • dblclick
    • mouseover
    • mouseout
    • mousemove
  • कीबोर्ड एवेंट्स:
    • keydown
    • keyup
    • keypress
  • फॉर्म एवेंट्स:
    • submit
    • change
    • focus
    • blur
