JavaScript बेसिक्स

1. JavaScript क्या है?

JavaScript एक प्रोग्रामिंग भाषा है जो वेब डेवलपमेंट के लिए बहुत महत्वपूर्ण है। यह एक डायनामिक और फ्लेक्सिबल भाषा है जो:

  • वेब पेजेज को इंटरैक्टिव बनाती है
  • यूजर इनपुट को हैंडल करती है
  • डेटा को मैनिपुलेट करती है
  • API कॉल्स करती है

2. वेरिएबल्स और डेटा टाइप्स

वेरिएबल्स डिक्लेरेशन:

// let का उपयोग
let name = "John";

// const का उपयोग
const age = 25;

// var का उपयोग (पुराना तरीका)
var isStudent = true;

डेटा टाइप्स:

  • String: टेक्स्ट डेटा
  • Number: नंबरिक डेटा
  • Boolean: true/false
  • Array: लिस्ट ऑफ वैल्यूज
  • Object: की-वैल्यू पेयर्स
  • null: खाली वैल्यू
  • undefined: अनडिफाइंड वैल्यू

उदाहरण:

// String
let message = "Hello World";

// Number
let number = 42;

// Boolean
let isActive = true;

// Array
let fruits = ["Apple", "Banana", "Orange"];

// Object
let person = {
    name: "John",
    age: 30,
    isStudent: false
};

3. ऑपरेटर्स

मुख्य ऑपरेटर्स:

  • अरिथमेटिक: +, -, *, /, %
  • कम्पेरिजन: ==, ===, !=, !==, >, <, >=, <=
  • लॉजिकल: &&, ||, !
  • असाइनमेंट: =, +=, -=, *=, /=

उदाहरण:

// अरिथमेटिक ऑपरेटर्स
let a = 10;
let b = 5;
console.log(a + b); // 15
console.log(a - b); // 5
console.log(a * b); // 50
console.log(a / b); // 2

// कम्पेरिजन ऑपरेटर्स
console.log(a > b);  // true
console.log(a === b); // false

// लॉजिकल ऑपरेटर्स
console.log(true && false); // false
console.log(true || false); // true

4. कंट्रोल स्ट्रक्चर्स

if-else स्टेटमेंट:

if (condition) {
    // कोड
} else if (anotherCondition) {
    // कोड
} else {
    // कोड
}

लूप्स:

// for लूप
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// while लूप
while (condition) {
    // कोड
}

// do-while लूप
do {
    // कोड
} while (condition);

उदाहरण:

// if-else उदाहरण
let age = 18;
if (age >= 18) {
    console.log("वोट कर सकते हैं");
} else {
    console.log("वोट नहीं कर सकते");
}

// for लूप उदाहरण
for (let i = 1; i <= 5; i++) {
    console.log(`संख्या: ${i}`);
}

5. फंक्शंस

फंक्शन डिक्लेरेशन:

// बेसिक फंक्शन
function add(a, b) {
    return a + b;
}

// एरो फंक्शन
const multiply = (a, b) => a * b;

// एनॉनिमस फंक्शन
const divide = function(a, b) {
    return a / b;
};

उदाहरण:

// फंक्शन उदाहरण
function greet(name) {
    return `नमस्ते ${name}!`;
}

// एरो फंक्शन उदाहरण
const square = (num) => num * num;

console.log(greet("राहुल")); // नमस्ते राहुल!
console.log(square(5)); // 25
नोट: यह एक बेसिक ट्यूटोरियल है। आगे के टॉपिक्स जैसे DOM मैनिपुलेशन, एरर हैंडलिंग, प्रॉमिसेज, और एसिंक्रोनस प्रोग्रामिंग के लिए अगले पेजेज देखें।
